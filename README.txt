SFDEsim

SFDEsim is a Python based simulator prototype made for simulating simple electrical or mechanical systems.
The simulator assits with the visualization and simulation control, where as the simulations are programmed in Python in a separate simualtion file.
The program is built using PySide6 and PyQtGraph libraries.

This project was created as a part of a Master's thesis project for electrical engineering degree.
https://urn.fi/URN:NBN:fi-fe20241213102328

For end user, see manuals in the Documents directory.
